If $\mu$ and $\sigma^2$ represent mean and variance of $X$, respectively, then $10(\mu^2 + \sigma^2)$ is equal to

  • $250$

Question 70

Consider the following system of equations

$$ \begin{align*} \alpha x + 2y + z = 1 \\ 2\alpha x + 3 y + z = 1 \\ 3x + \alpha y + 2z = \beta \end{align*} $$ for some $\alpha, \beta \in \mathbb{R}$. Then which of the following in NOT correct.

  • It has no solution if $\alpha = -1$ and $\beta \ne 2$
  • It has a solution for all $\alpha \ne -1$ and $\beta = 2$
  • It has no solution for $\alpha = -1$ and for all $\beta \in \mathbb{R}$
  • It has no solution for $\alpha = 3$ and for all $\beta \ne 2$

  • $\Large{\frac{9}{8}}$
  • $\Large{\frac{3}{2}}$
